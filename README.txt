GESTIONALE FRUTTA E VERDURA DI CORATELLA

COSA FA
- Dashboard generale
- Anagrafica clienti
- Scheda cliente con storico ordini e totale speso
- Inserimento nuovi ordini
- Stato ordine: Da preparare / In consegna / Consegnato / Annullato
- Agenda ordini per data
- Impostazioni attività
- Backup esporta/importa JSON
- Logo già inserito in SVG modificabile

FILE DA MODIFICARE SOLO SE SERVE
- data/db.json -> per mettere numero, WhatsApp, email reali
- public/logo.svg -> per sostituire il logo con quello ufficiale

COME AVVIARLO IN LOCALE
1) Apri Terminale dentro la cartella del progetto
2) Esegui:
   npm install
   npm start
3) Apri nel browser:
   http://localhost:3000

PER METTERLO ONLINE
Va bene su:
- Render
- Railway
- VPS con Node.js

DATI DA INSERIRE SUBITO
Apri /settings nel browser e aggiorna:
- Telefono
- WhatsApp
- Email
- Indirizzo
